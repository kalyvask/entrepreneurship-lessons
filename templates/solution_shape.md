# Solution Shape — Template

Used in [Stage 03 — Problem-Solution Fit](../stages/03_problem_solution_fit.md). A solution shape is **not** a feature list and **not** a spec. It's the rough functional answer to a validated problem — concrete enough to evaluate, abstract enough that you're not committing to specifics yet.

Sketch 2–3 candidate shapes. Test them as concept tests (see [`playbooks/concept_test.md`](../playbooks/concept_test.md)) before picking one.
