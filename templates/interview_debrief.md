---

## Desperation scorecard

Mark each marker the interviewee exhibited:

- [ ] **Would pay with little proof** — would buy on POC, doesn't need 50 references
- [ ] **Has tried to build / hack / work around the problem** — spreadsheet, internal tool, consultant, contractor
- [ ] **Mentioned the pain unprompted** — brought it up themselves, didn't need me to surface it
- [ ] **Wants it now, not next quarter** — "when can I have it?" not "let's review in Q3"

**Score:** ___ / 4

- 3–4 = desperate customer. High signal.
- 1–2 = needy. Don't build for them yet.
- 0 = polite. Probably wrong segment.
