# PMF Dashboard

The only question this answers: **are the dogs eating the dog food?** It tracks the six
signals from the Stage 06 rubric, measured **in your narrow *who*** — not across all signups,
not across everyone you can get to pay.

Updated: ____  |  Narrow *who*: ____  |  Active customers in the narrow *who*: ____

| Signal | Current | Bar | Status |
|---|---|---|---|
| **Sean Ellis** ("very disappointed" if it went away) | __% | ≥ 40% in the narrow who | |
| **Retention curve** (weekly, by cohort) | shape: ____ | flattens above zero, improving cohort-over-cohort | |
| **Organic acquisition** (% of new customers) | __% | > 40% and climbing | |
| **Leap of faith** verified by behaviour | ____ | a specific, repeatable behaviour | |
| **The tells** (referrals unasked, shrinking cycle, pricing power) | ____ | 3+ present | |
| **Held the narrow who** (didn't broaden to anyone who'd pay) | ____ | yes | |

**PMF =** all ≥ bar, AND Sean Ellis ≥ 40% with retention flattening. Below that you do not
have PMF, regardless of revenue.

## What this dashboard refuses to track (vanity)

Listed so they don't sneak in as "progress":

- Total signups / waitlist size (signups ≠ usage)
- Total revenue without organic % and retention (can be paid-acquired with leaky retention)
- Press coverage, awards, investor interest (narrative, not desperation)
- Geographic spread / number of countries (breadth ≠ depth)
- One big customer who "loves us" (one advocate ≠ a market)
